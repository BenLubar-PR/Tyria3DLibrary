HavokRenderer Class

Extends DataRenderer
Defined in: src\dataRenderer\HavokRenderer.js:3
Module: T3D

A renderer that generates meshes describing the collisions of a map.

Constructor

HavokRenderer

(
  • localReader
  • settings
  • context
  • logger
)

Defined in src\dataRenderer\HavokRenderer.js:3

Parameters:

  • localReader LocalReader

    The LocalReader instance to read data from.

  • settings Object

    Any settings used by this renderer. Must specify "mapFile", a GW2File. If "visible" is specified and true, the generated meshes will be textured with a MeshNormalMaterial, otherwise they will not be visible.

  • context Object

    Shared value object between renderers.

  • logger Logger

    The logging class to use for progress, warnings, errors et cetera.
