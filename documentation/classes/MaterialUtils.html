getMaterial

(
  • material
  • materialFile
  • localReader
  • sharedTextures
)
THREE.Material

Defined in src\util\MaterialUtils.js:153

Builds a THREE texture from a ModelMaterialData by reading settings and loading any required data from the localReader. Uses sharedTextures for texture caching.

This method is full of guesses and estimations, and could be improved on a lot, allowing rendering of multi UV channeled materials, or special materials like custom color chanelled gear.

Parameters:

  • material ModelMaterialData

    A value object often automaticaly
    generated by a GW2Chunk structure definitions can be found in AllFormats.js, look for the latest version of ModelMaterialData

  • materialFile GW2File

    A GW2File instance, must be of type AMAT

  • localReader LocalReader

    The LocalReader to load the file contents from.

  • sharedTextures Object

    Value Object for keeping the texture cache

Returns:

THREE.Material:

A THREE Material with the generated contents and settings.
